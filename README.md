## 🌍 TimeMapper

Conversor de fusos horários desenvolvido com Next.js, TypeScript e Tailwind CSS, permitindo a conversão rápida entre diferentes localidades do mundo através de pesquisa inteligente de cidades e identificação automática de timezones.

## 📷 Preview

## 🚀 Funcionalidades

- Conversão de horários entre diferentes fusos horários.
- Pesquisa de cidades em tempo real.
- Autocomplete de localidades utilizando GeoNames.
- Identificação automática do timezone da cidade selecionada.
- Conversão bidirecional entre origem e destino.
- Interface responsiva para desktop, tablets e smartphones.
- Splash Screen personalizada.

Suporte multilíngue:
- Português
- Inglês
- Espanhol

- Botão para limpar campos.

## ⚙️ Como Executar o Projeto:

1º - Clone o repositório
git clone URL_DO_REPOSITORIO

2º - Instale as dependências
npm install

3º - Configure as variáveis de ambiente

Crie um arquivo .env.local na raiz do projeto:

GEONAMES_USERNAME=seu_usuario_geonames

Execute o projeto:
npm run dev

Acesse:
http://localhost:3000
